Developer Biography

Hey, I'm Callum. I'm a 24-year-old developer living in Scotland. My path through software engineering focuses deeply on understanding how virtual environments handle complex logic, ticking pipelines, and raw execution speeds. I study Minecraft's Java runtime as an environment for performance engineering, aiming to build tools that maximize structural efficiency while keeping gameplay systems complex and engaging.

Tourette's Syndrome & The Coding Outlet

One of the key aspects of my life is navigating Tourette's Syndrome. While it brings daily challenges, I have found that coding acts as a powerful focusing outlet. The moment I begin writing Java systems, mapping data loops, or investigating error stack traces, my brain enters a state of deep hyper-focus. The structured and absolute logic of software engineering provides a stabilizing focus where my tics quiet down, transforming a challenging condition into an engine for meticulous, detail-oriented work.

My deep love for Minecraft is what drives me to write mods. I don't want to just create random features; I want my passion for the game to show in the quality, structure, and execution of my work. I treat mod development not as a casual add-on, but as a discipline where code quality directly shapes the player's experience. This drives my interest in modern architectures like NeoForge, aiming to deliver mods that run seamlessly even in high-load setups.

System Tinkering & Technical Toolkit

When I'm not debugging Forge registries or writing clean tick-rate optimizations, I'm almost certainly tinkering with other tech-oriented hobbies. I'm driven by a curiosity to dissect systems, explore alternative environments, and optimize hardware configs.

System Customization

I enjoy experimenting with configuration scripts, exploring command-line tools, tuning operating system behaviors, and tweaking Linux and shell setups to streamline development tasks.

Java Workflows

I focus my engineering exclusively on Java, using automation scripts to manage git tasks, parse files, and optimize runtime performance pipelines.
