Biography

Hey, I'm Callum. I'm a 24-year-old developer living in Scotland. I write mods for Minecraft, mainly targeting the Java environment using NeoForge. I like looking at how virtual environments handle automation and mechanical logic, aiming to build tools that keep gameplay engaging without dragging down performance.

Tourette's & Coding

Dealing with Tourette's Syndrome means managing physical and vocal tics throughout the day. It takes a lot of energy, but programming has been a game-changer for me. When I'm coding in Java or debugging scripts, my mind enters a state of absolute focus. This structured logic helps calm my tics down, acting as a quiet and steady environment where I'm completely in control of what I'm creating.

I have a deep love for Minecraft, and my goal is to make sure my passion for the game shows in my mods. I want to build things that feel cohesive and run smoothly. For me, mod development is about creating reliable, high-quality additions to the game that add fun new elements while maintaining good performance.

Tech Tinkering

When I'm not modding, I'm usually tinkering with other tech projects. I like configuring local setups, writing scripts, and playing around with system customization.

System Customization

I enjoy writing command-line configurations, customizing macOS shell terminals, and tweaking local Linux setups to streamline my dev workflow.

Java Workflows

I focus my coding exclusively on Java, using simple scripts to automate git tasks, parse mod assets, and test performance loops.
